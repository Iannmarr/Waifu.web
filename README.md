El proyecto Waifu2X Web consiste en el desarrollo de una interfaz web (Landing
Page) moderna y responsiva. Su propósito principal es servir como punto de entrada
para la difusión, documentación y descarga del software "Waifu2X", una herramienta
basada en inteligencia artificial para el escalado de imágenes y reducción de ruido.
1.2. Objetivos del Sitio
● Conversión: Facilitar la descarga del software mediante llamadas a la acción
(CTA) claras.
● Información: Explicar de manera concisa el funcionamiento del algoritmo de
escalado.
● Navegación: Redirigir al usuario a secciones secundarias como Guías de
Uso, Descargas y la sección institucional (Nosotros).
